```Shell
├── FELT_SOT dataset

        ├── dvSave-2022_10_11_19_24_36
            ├── dvSave-2022_10_11_19_24_36_aps

            ├── dvSave-2022_10_11_19_24_36.aedat4

        ├── dvSave-2022_10_11_19_43_03

            ├── dvSave-2022_10_11_19_43_03_dvs.aedat4
```
